# ARCO — Company Website

Modern frontend website for **ARCO**, a software development company.

## Pages

- **Home** — Hero, services preview, process, testimonials, CTA
- **About** — Company story, values, stats
- **Services** — Full service offerings and workflow
- **Contact** — Form (sends via WhatsApp), phone, social links

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- React Router

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build for Production

```bash
npm run build
npm run preview
```

Deploy the `dist/` folder to any static host (Netlify, Vercel, GitHub Pages, etc.).

## Configuration

Edit `src/config.ts` for phone, WhatsApp, and social URLs.
